# 桃園機場捷運時刻表
## 2018/03/01

### 檔案格式： CSV

### 欄位說明

#### 編號
排序資料用。

#### 車站時刻
 * 採用 24 小時制表示，即 `HH:MM` 。若欄位為 `xxxxx` ，代表該列車表定並未通過該站；若欄位為 `--:--` ，表示該列車通過該站不停靠。
 * 終點站為推估到達時刻。

## 關於本時刻表
 * 本時刻表僅供參考，若和[官方時刻表](https://www.tymetro.com.tw/tymetro-new/tw/_images/document/travel-guide/time0224.pdf)有所出入，請以官方時刻表為準。
